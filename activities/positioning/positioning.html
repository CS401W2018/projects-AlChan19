Positioning

  1. What is normal flow in HTML? - It is the default way elements appear on a webpage.
  2. What does float do? Why do you need to clear the float? - Float moves elements and clearing float prevents layout issues.
  3. What do absolute, fixed, and relative do? - Absolute fixes an element in place, fixed keeps it static, and relative moves it slightly.
  4. How can you make nice navigation buttons for mobile devices? - Use large buttons, space them well, and consider a hamburger menu.
