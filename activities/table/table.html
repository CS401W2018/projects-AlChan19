Engagement – Tables

  1. When do you use a table?
    • You use a table when you need to show information in rows and columns.
    • Use it when you want to organize data like schedules, prices, or grades.
    • Tables are good for comparing information side by side.
  2. When do you not use a table?
    • Do not use a table to design the layout of your web page.
    • Avoid using tables to place images, buttons, or text in a certain position.
    • Tables should only be used for showing data, not for page design.
  3. When do you need to do extra work for table accessibility?
    • You need extra work when you want everyone, including people with disabilities, to understand your table.
    • Add table headers (th) to label the data in each column or row.
    • Use the scope attribute (scope="col" or scope="row") to make it clear which data belongs to which header.
