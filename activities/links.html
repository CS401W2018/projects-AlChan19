Questions

  1. Explain the difference between an absolute link and a relative link. Why would you use relative links?

    2. An absolute link is a full address that takes you to a specific website (e.g., https://example.com). A relative link is a shorter address that works based on where your file is (e.g., sample.html). You use relative links to keep things simple and to make your links work easily within the same site.

  2. List the 3 rules with relative links.
    • Use the file name if the linked file is in the same folder (e.g., page.html)
    • Use ../ to move up one folder (e.g., ../page.html)
    • Use the folder name starting from the root to link to a file in a different folder (e.g., folder/page.html)
