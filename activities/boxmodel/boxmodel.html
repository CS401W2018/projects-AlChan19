Box Model

  1. Block tags create a new section on the page and take up the full width, like paragraphs or headings. Inline tags only take up as much space as needed, like links or bold text.
  2. The margin is the space outside an element. the border is the edge around the element. the padding is the space between the border and the content inside.
  3. You can use different units like pixels(px), percentages(%), em, rem, centimeter(cm), millimeter(mm), inches(in), and points(pt).
  4. Use it when adding images,charts, illustrations with a caption. It helps group the image and its description together.
