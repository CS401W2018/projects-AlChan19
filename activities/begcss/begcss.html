CSS

Questions:

  1. Difference between inline, embedded, and external CSS:
    • Inline CSS – written inside an HTML tag using the style attribute. It has the highest priority and takes effect over both embedded and external CSS.
    • Embedded CSS – placed inside a style tag in the head section of the HTML file. It has medium priority and applies instead of external CSS but not CSS inside a tag.
    • External CSS – written in a separate .css file and linked to the HTML file. It has the lowest priority but is best for keeping styles organized.
  2. Why use only external CSS?
    • External CSS is the best because it keeps styles in one file, making the code cleaner, cohesive and easier to update. It also helps when using the same styles on many pages.
  3. What is a selector used for?
    • A selector in CSS is used to choose which HTML elements to style. For example, p { color: red; } makes all elements red.
