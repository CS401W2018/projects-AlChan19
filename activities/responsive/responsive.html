Engagement - Page for Print

  1. Why is the viewport meta tag important?
    • It helps websites look good on different screen sizes.
    • It makes sure the page scales correctly on mobile devices.
    • Without it, the website might not fit well on small screens.
  2. How can you adjust the CSS for printing?
    • Use a separate CSS file with 'media="print".'
    • Hide elements that are not needed for printing, like navigation bars.
    • Use black-and-white colors to save ink.
  3. When would you want to adjust the CSS for printing?
    • When you want a webpage to print neatly without extra elements.
    • When printing articles, receipts, or important documents.
    • To make sure the printed page is easy to read.
