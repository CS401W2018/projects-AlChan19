Internet

The internet is a global network of computers and electronic devices that are connected all over the world to communicate with each other. INTERNET stands for Interconnected Network, providing a variety of information and communication methods for machines and humans using it.

Web

The web stands for World Wide Web (WWW or W3). It refers to all public websites or pages that run on top of the internet. It provides users with access to a wide range of content and data.

How are the Internet and Web different?

The computer is a network of computers that are linked across the world that allows access to the World Wide Web for data to be transferred. While the web is the collection of data that runs on the internet.

ASCII and Unicode

ASCII (American Standard Code for Information Interchange) and Unicode are encoding standards that represent text and characters on computers and other devices. ASCII has 128 characters that primarily covers English letters, numbers, and symbols, while Unicode has 140,000 characters, including emojis and other special symbols.

What is UTF-8

UTF-8 stands for Unicode Transformation Format 8. It is a character encoding standard that represents characters using one to four bytes. It is the World Wide Web's most common character encoding.

URL

URL stands for Uniform Resource Locator. It is a web address that is the location of a specific file or webpage on the internet. It tells web browsers how and where to retrieve a web resource.

Domain Name

A domain name refers to website address. It is typed into a browser window to reach a particular website. It is used by individuals, businesses, organizations, and government agencies.

HTTP

HTTP stands for Hypertext Transfer Protocol. It is the foundation of the World Wide Web. It is used to transfer data and load web pages over a network. It sends information between the web browser and the web server.

HTTPS

HTTPS stands for Hypertext Transfer Protocol Secure. It is an extension of HTTP and is more secured. It uses encryption for secure communication over a computer network.

IP Address

An IP (Internet Protocol) address is a unique address that tells you the location of a device or network, and identify the network owner.

DNS

DNS stands for Domain Name System. It turns domain names into IP addresses so browsers can load internet resources. DNS makes it easier for users to access the internet without having to memorize IP addresses for every website.

W3C Recommendations

W3C Recommendations are rules and guidelines from the World Wide Web Consortium (W3C). They help make web technologies work well together and make sure that websites are consistent and easy to use across the Internet.

XML, XHTML, HTML, HTML5

XML stands for Extensible Markup Language is used to share information between websites, apps, and data. XHTML stands for Extensible HyperText Markup Language is a stricter version of HTML that follows XML rules. HTML stands for Hypertext Markup Language is the main language for creating web pages and is the latest version of HTML and it improves multimedia content.
